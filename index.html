Setup Instructions

1. How to use with LiveContainer (via LcInstaller)

The simplest way to install apps from this repo directly into LiveContainer:

  1. Sideload and open LcInstaller inside your device or LiveContainer.
  2. Tap the "Sources" or "+" button in LcInstaller.
  3. Add this Repository URL: https://aoi.github.io/aoirepo/apps.json
  4. Browse and tap install directly into LiveContainer.

2. How to use with AltStore / SideStore

Register the source to install apps directly:

  1. Click the "Add to AltStore" or "Add to SideStore" button at the top of this page.
  2. If prompted, open in AltStore/SideStore.
  3. The source will be automatically added to the Sources tab, and you can now download and update apps.
